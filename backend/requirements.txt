=======================================================
CORE WEB FRAMEWORK AND ASYNC SERVER
=======================================================
fastapi

gunicorn

Uvicorn (standard includes necessary speedups like httptools)
uvicorn[standard]

=======================================================
DATABASE (POSTGRESQL & ORM)
=======================================================
SQLAlchemy (Core ORM)
SQLAlchemy

Async driver for PostgreSQL (best compatibility with FastAPI)
asyncpg

Pydantic (used for data validation and schema definition, auto-included in FastAPI, but listed for clarity)
pydantic

Recommended library for loading environment variables (e.g., .env file)
python-dotenv

=======================================================
AUTHENTICATION AND SECURITY
=======================================================
Library for encoding/decoding JWTs
python-jose[cryptography]

Library for secure password hashing (bcrypt)
passlib[bcrypt]

=======================================================
CACHING AND ASYNC MESSAGING
=======================================================
Redis client (supports asyncio) for caching and session management
redis

Celery (distributed task queue)
celery

Message broker for Celery (using Redis as the broker)
Redis is often used as both the broker and backend for Celery
If using Kafka as the Celery broker, replace with 'kombu-kafka' or similar package
kombu

Kafka client for producing/consuming messages across microservices
confluent-kafka # or python-kafka, confluent-kafka is often preferred

elasticsearch
